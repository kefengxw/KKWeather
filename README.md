# KKWeather

This a weather Android project which is based on Kotlin.

Features of Android:
1. RecyclerView,
2. Acitivity, Intent, Layout,
3. Fragment,
4. SharedPreferences, Database,
5. okHttp3, GSON,
6. Service in background, IntentService,
7. DrawerLayout,
8. SwipeRefreshLayout,
